# TraceBack — Smart Campus Lost & Found Platform

## Overview

TraceBack is a full-stack platform designed to help students and staff report and recover lost items efficiently within a campus environment.

## Features

* Secure user authentication
* Lost item reporting
* Claim management workflow
* Item search and filtering
* Role-based access (Admin / User)

## Tech Stack

Frontend:

* React.js

Backend:

* Spring Boot
* REST APIs

Database:

* PostgreSQL

Other Tools:

* Docker
* JWT Authentication
* Render Cloud Deployment

## System Architecture

1. Users report lost items through the frontend
2. Backend APIs process item data
3. PostgreSQL stores item records
4. Claims verified through admin workflow

## Installation

### Backend

```bash
mvn spring-boot:run
```

### Frontend

```bash
npm install
npm start
```
